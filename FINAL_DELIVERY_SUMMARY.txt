GETTING STARTED
================================================================================

1. Prerequisites:
   - Node.js 18+ installed
   - PostgreSQL 12+ installed
   - pnpm or npm available

2. Quick Setup (5 minutes):
   $ cd /vercel/share/v0-project
   $ pnpm install
   $ psql -U postgres -c "CREATE DATABASE security_testing;"
   $ echo 'DATABASE_URL="postgresql://postgres@localhost:5432/security_testing"' > .env.local
   $ echo 'JWT_SECRET="change-this-in-production"' >> .env.local
   $ pnpm prisma migrate dev --name init
   $ pnpm dev

3. Access the Application:
   - Open http://localhost:3000 in browser
   - Click "ENTER TESTING FRAMEWORK" button
   - Start typing curl commands in terminal
   - Or click template buttons to load examples

4. Next Steps:
   - Read COMPLETE_BUILD_GUIDE.md for detailed setup
   - Read API_DOCUMENTATION.md for API reference
   - Read DATABASE_SETUP.md for PostgreSQL setup
   - Test vulnerabilities in baseline API
